
# Movie notes API 

About:

This is the api responsible for create, update, delete users end notes in the Rocketmovie application


## Stacks

 Node, Express, nodemon, knex, sqlite, sqlite3, bcryptjs, express-async-errors, nodemon



## Running locally

clone the project

```bash
  git clone https://github.com/Italovini223/movies_notes_API.git
```


nstall the dependencies

```bash
  npm install
```

Start sever

```bash
  npm run dev
```
